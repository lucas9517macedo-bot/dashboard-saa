Como escalar, conectar e publicar o painel SAA
📊 Estrutura Excel/CSV
Cada coordenação preenche uma aba padrão:
// Colunas obrigatórias
coordenacao | kpi_id | kpi_nome
realizado | meta | unidade
inverso | semana | fonte

// Exemplo
CGOF | FIN-01 | Execução Orçam.
67 | 80 | %
false | 2025-W42 | SIAFI
⚡ Integração JSON / API
Conecte qualquer fonte de dados:
// Carregar dados externos
fetch('data.json')
 .then(r => r.json())
 .then(d => {
  Object.assign(DB, d);
  renderAll();
 });
🌐 Publicar no GitHub Pages
Deploy gratuito em minutos:
1
Crie repositório saa-dashboard
2
Suba o HTML como index.html
3
Settings → Pages → Source: main
4
URL: usuario.github.io/saa-dashboard

Modo Visualizar — acesso somente leitura para qualquer gestor. Senha padrão: saa2025

Modo Editar — acesso completo para alta gestão. Senha padrão: saa@gestao

Para alterar as senhas, edite as constantes PASS_VIEW e PASS_EDIT no código JS.
